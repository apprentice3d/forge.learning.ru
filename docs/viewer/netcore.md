# Client-side files (.NET Core)

Typically, ASP.NET Core app serves files from `/wwwroot/` folder, create it if not there yet. Under that folder, let's organize like this:

- `wwwroot/`: `.html`
- `wwwroot/js`: `.js`
- `wwwroot/css`: `.css`
